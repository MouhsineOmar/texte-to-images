# Image Generation Frontend

Cette application est le frontend React pour le générateur d'images.

## Installation

1. Installez les dépendances :
   ```bash
   pnpm install
   ```

2. Configurez l'URL de l'API dans un fichier `.env` :
   ```bash
   cp .env.example .env
   ```

3. Lancez le serveur de développement :
   ```bash
   pnpm dev
   ```

## Technologies
- React 19
- Vite
- Tailwind CSS
- Lucide React
- Sonner (notifications)
